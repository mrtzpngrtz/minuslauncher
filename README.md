# minuslauncher

A minimal, text-based Android launcher designed to reduce distractions and promote intentional phone use.

## Features

- **Minimalist Design**: A clean, text-only interface that keeps your home screen clutter-free.
- **Intentionality**: Select only the apps you need to see on your home screen.
- **Clock & Date**: Simple display of time and date.
- **Notification Indicators**: Subtle visual cues (a dot/colon) for apps with active notifications.
- **Dark Mode Support**: Built with Material 3 and follows system theme.
- **Custom Font**: Uses Hanken Grotesk for a clean, modern look.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mrtzpngrtz/minuslauncher.git
   ```
2. Open the project in Android Studio.
3. Build and run the app on your Android device.

## Usage

1. Set **minuslauncher** as your default home app.
2. Tap the **Settings** (gear) icon in the top right.
3. Select the apps you want to appear on your home screen.
4. Enable **Notification Access** when prompted to see notification indicators.

## Privacy

minuslauncher does not collect any personal data. It requires Notification Access only to display indicators on the home screen.
